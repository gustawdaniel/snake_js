# snake_js
Snake game written in javascript using objects. 


# Instaltion

To install dependencies

    yarn install

To run

    node node_modules/http-server/bin/http-server 

# Game

To game run

    firefox localhost:8080
    
Pres space to start and use arrows to control snake.
